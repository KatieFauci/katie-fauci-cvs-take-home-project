# katie-fauci-cvs-take-home-project

## Start API

set `API_READ_ACCESS_TOKEN` in `.env` to your bearer token

**Run:**
```
npm run start
```

or 

```
npm run build
npm run serve
```

** `npm run dev` is also included

## Example Request: 
`http://localhost:3000/MoviesByYear?year=2019` 


## Run Unit Tests
`npx ts-node unitTests.ts`
